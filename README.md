# Nashville Housing Data Cleaning with SQL

## Project Overview
This project demonstrates how SQL Server can be used to clean and prepare raw housing data for analysis.

## Skills Used
- SQL Server
- Data Cleaning
- Joins
- CASE Statements
- Window Functions
- Common Table Expressions (CTEs)
- ALTER TABLE
- UPDATE Statements

## Data Cleaning Tasks
- Standardized date formats
- Populated missing property addresses
- Split property and owner addresses into separate columns
- Converted SoldAsVacant values
- Identified duplicate records
- Removed unnecessary columns
